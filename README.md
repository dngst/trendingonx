## Requirements

* Ruby

* Bundler

* Ruby on Rails

* PostgreSQL

See [Gemfile](./Gemfile)

## Install

$ bin/setup

$ bin/dev

## Caching

Run $ bin/rails dev:cache to toggle caching

## Tests

$ bundle exec rspec
